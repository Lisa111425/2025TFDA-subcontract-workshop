# 114年度食品藥物管理署醫療器材委託製造線上申辦系統審查人員教育訓練

##AI 技術應用於醫療器材委託製造許可(AI代理人工作坊)
<img width="1402" height="200" alt="image" src="https://img.freepik.com/free-vector/bokeh-lights-glitter-background_1048-8548.jpg?semt=ais_hybrid&w=740&q=80" />

活動目標
參與者學習使用 AI 代理從模擬醫療器材提交材料中提取關鍵資訊，如裝置規格、臨床數據、製造細節。重點在修改提示詞以優化提取準確性。
適用參與者
20 名 TFDA 官員，分組 4-5 人。
所需材料

Hugging Face Spaces AI 系統。
模擬文件：mock-device-submission1.pdf（包含裝置描述、規格表）。

步驟

介紹（5 分鐘）：講師解釋代理設定標籤，示範預設系統提示。
實作（20 分鐘）：
上傳 mock-device-submission1.pdf 並執行 OCR（頁碼 1-10，語言：英文）。
在代理設定中，選擇第一代理（醫療器材規格提取器）。
修改系統提示：新增「以 JSON 格式輸出，包括裝置名稱、材料、尺寸」。
調整參數：溫度 0.1（更精確）、max_tokens 1500。
執行代理，檢視輸出。

優化與討論（10 分鐘）：
組內實驗不同提示詞（如新增「忽略非相關文字」）。
分享結果：哪個提示最準確？為何？

成果：每個組產生提取報告，下載 Markdown。

預期學習

理解提示工程對 AI 輸出的影響。
應用於真實審查：加速規格提取。


activities/activity4.md 內容
活動目標
參與者使用 AI 代理分析模擬醫療器材提交材料中的臨床試驗數據，提取療效指標、安全性數據，並評估統計顯著性。為了增加趣味性，設計成「挑戰賽」形式，組隊競爭誰能最快、最準確地優化 AI 輸出。
適用參與者
20 名 TFDA 官員，分組 4-5 人。
所需材料

Hugging Face Spaces AI 系統。
模擬文件：mock-device-submission4.pdf（包含臨床試驗報告、數據表）。

步驟

介紹（5 分鐘）：講師解釋臨床試驗代理設定，示範基本提示，並宣布規則：每組有 3 次嘗試機會，計分依準確度（80%）、速度（10%）、創意提示（10%）。
實作挑戰（20 分鐘）：
上傳 mock-device-submission4.pdf 並執行 OCR（頁碼 5-15，語言：英文）。
設定代理：使用「臨床試驗資料分析器」，修改提示詞如「提取主要終點、p 值，並計算效果大小（Cohen's d）」，但必須在提示中加入一個「趣味元素」（如比喻解釋統計）。
調整參數：溫度 0.3，max_tokens 1800，但挑戰：只能使用 Gemini 模型。
執行並優化：每組限時 15 分鐘內提交最佳輸出。

評比與討論（10 分鐘）：
講師評分並宣布贏家（獎勵：虛擬徽章或小禮物）。
組內討論：哪個提示最有效？挑戰如何影響結果？

成果：每組產生臨床分析報告，下載 JSON 並分享最佳提示。

預期學習

掌握 AI 在臨床數據分析的應用。
透過遊戲化，提升提示工程技能，並學習在限制下優化。


activities/activity5.md 內容
活動目標
參與者透過 AI 代理檢查醫療器材標籤是否符合 TFDA 要求（如 21 CFR Part 801），識別缺失，並生成改進建議。加入角色扮演元素，讓參與者「扮演」不同 TFDA 角色（如審查官、製造商），增加互動趣味。
適用參與者
20 名 TFDA 官員，分組 4-5 人。
所需材料

Hugging Face Spaces AI 系統。
模擬文件：mock-device-submission5.pdf（包含標籤設計、說明書）。

步驟

介紹（5 分鐘）：講師分配角色（例如：2 人審查官、2 人製造商、1 人仲裁者），解釋遊戲規則：審查官用 AI 找缺失，製造商辯護，仲裁者決定。
實作遊戲（20 分鐘）：
上傳 mock-device-submission5.pdf 並 OCR（全頁，語言：繁體中文）。
設定代理：使用「藥品標籤與說明書檢查器」，修改提示詞如「檢查標籤是否包含必要警語，並模擬使用者誤讀風險」。
調整參數：溫度 0.4（增加創意建議），但挑戰：必須整合前一活動輸出作為輸入（管道模式）。
角色互動：審查官執行 AI，製造商提出反駁提示修改，限 3 輪辯論。

決勝與討論（10 分鐘）：
仲裁者宣布結果，贏家組獲得「最佳審查團隊」稱號。
全體討論：角色扮演如何模擬真實審查流程？

成果：產生標籤審查報告 Markdown，下載並記錄辯論要點。

預期學習

理解 AI 在標籤合規的角色。
透過角色扮演，增強團隊合作與批判思考，樂趣中學習挑戰
