# 114年度食品藥物管理署療器材委託製造線上申辦系統審查人員教育訓練
# AI 技術應用於醫療器材委託製造許可：AI代理人工作坊

活動目標：
參與者學習使用 AI 代理從模擬醫療器材委託製造許可資料中提取關鍵資訊，例如醫療器材商名稱、委託製造品項等資訊。使用者透過優化AI Agent提示詞以提升準確性。
適用參與者：
20 名 TFDA審查人員。
所需材料：
Tfda agentic AI代理人輔助審查系統(將於活動中提供系統網址)
模擬申請文件：mock-submissions(請於/mock/資料夾中取得)

活動步驟：
活動說明與系統介紹（5 分鐘）：講師介紹Tfda agentic AI代理人輔助審查系統，並示範如何設定代理人與提示詞。
實作（20 分鐘）：
上傳模擬申請文件，並執行 OCR（頁碼，語言：英文）。
選擇AI Agent。
修改系統提示：。
調整參數：溫度 0.1（更精確）、max_tokens 1500。
執行代理，檢視輸出。

系統說明(Tfda agentic AI代理人輔助審查系統)：
系統需求：使用 Chrome 或 Edge 等瀏覽器。
帳號：免帳號直接使用
語言：預設繁體中文，可切換英文。
系統設定：可由側邊欄設定系統個人偏好設定。
主要功能：提供pdf檔案OCR識別、Agentic AI文件內容探勘、保留審查筆記(紀錄)
資訊安全：系統不保留任何資料，瀏覽器關閉即清除所有暫存資料

系統說明操作步驟：
活動步驟1：模擬文件(pdf)上傳
活動步驟2： 預覽與編輯 OCR 文字
活動步驟3：AI Agent代理人設定 (提示詞與選擇LLM模型)
活動步驟4：執行AI Agent代理人分析，優化提示詞以提升結果
活動步驟5：編輯審查筆記：支援 Markdown（如 # 標題、- 清單）和 HTML 顏色（如 紅色）。

挑戰活動：
挑戰項目一、
	任務目標：請將所提供模擬QMS核備函及模擬委託製造契約，利用AI Agent取得審查重點資訊(例如製造廠名稱、地址、品項、作業活動、分類分級品項代碼等)。
	任務材料：模擬QMS核備函、模擬委託製造契約。
挑戰項目二、
	任務目標：請透過AI Agent，針對模擬委託製造契約，輔助判定是否符合委託契約應包含內容。
	任務材料：委託契約審查重點、模擬委託製造契約。
